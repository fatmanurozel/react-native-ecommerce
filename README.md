# react-native-ecommerce

## Bilgisayarınızda Çalıştırın

Projeyi klonlayın

```bash
  git clone https://github.com/fatmanurozel/react-native-ecommerce
```

Proje dizinine gidin

```bash
  cd react-native-ecommerce
```

Gerekli paketleri yükleyin

```bash
  yarn
```
```bash
  yarn add formik
```
```bash
  yarn add formik yup
```

Sunucuyu çalıştırın

```bash
  yarn start
```
